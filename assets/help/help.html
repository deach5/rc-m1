The Menu

Help icon

Clicking the Help button opens this screen, to learn about navigation and interaction.

Closed Captions icon

By clicking ‘Close Captions’ this will show/hide the transcript for each screen with audio/video.

Menu icon

The Menu icon or ‘Hamburger button’ show/hides the menu bar items.

Exit icon

The Exit button saves the status of your current progress, and exits the module.

Underlined Menu item

Indicates the topic you are currently viewing.

Disabled Menu item

Indicates an unavailable topic. You will need to complete all topics proceeding it first, to gain access.

Pink Menu item

Indicates a completed topic. The topic is now unlocked, and you can freely navigate it at any time.

Course navigation and interaction

Next section icon

The down arrow will scroll to the next section in topic.

Note:
The arrow will be disabled until all activities in the current section are completed. This includes viewing videos, listening to narration, and completing interactions.

Video icon

Clicking this icon will play a video.

Video control

Some browsers allow you to maximise the video screen, clicking the icon will expand the video.

Download icon

Clicking the ‘Download icon’ will download relevant information to the topic, normally in a PDF format.

Watch – Listen – Engage

Clicking the ‘Watch – Listen – Engage’ will present information in numerus ways.

Note:
You will have to click this to enable the next section.

Module complete icon

The ‘Module Completion’ button will appear on the last section of the module, allowing you to save and exit module.
